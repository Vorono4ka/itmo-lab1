# Окружность

## Функции

- `area(r: float) -> float`

  Рассчитывает площадь окружности из её радиуса.

    - r - Радиус окружности

  Возвращает площадь окружности

- `perimeter(r: float) -> float`:

  Рассчитывает периметр окружности из её радиуса.
    - r - Радиус окружности

  Возвращает Периметр окружности
